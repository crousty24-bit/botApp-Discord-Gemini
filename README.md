# Discord Bot Setup

A Discord bot project for integrating with external services and APIs.

## Quick Start

1. Install dependencies:
```bash
npm install
```

2. Configure environment variables in `.env`

3. Run the bot:
```bash
node src/index.js
```

## Project Structure

- `src/index.js` - Main entry point
- `src/gemini.js` - Gemini API integration
- `src/webhook.js` - Discord bot communication
